# teamscli

cool project I made

## Usage

Help

```bash
teamscli --help
```


This will update your refresh token with a new one

```bash
teamscli refresh --tenant-id aaaaaaaa-d714-4a1f-8101-eeeeeeeeeeee --refresh-file <file with refresh token>
```
